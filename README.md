For full explanation for this project is inside each folder within this repository, but for put it simple this TO-DO project are using Next.js (Front-End) n Node.js (ECMAScript), Express.js, Sequelize, SQLite (Back-End).

## HOW TO RUN THIS PROJECT?
-- USE BUN --
1. bun install (to install dependencies for each folder)
2. bun run build (to build this project) (for backend there are other step to fully run this project, REAME already provide inside backend folder)
3. bun run dev (to run this project)

-- USE NPM --
1. npm install (to install dependencies for each folder)
2. npm run build (to build this project) (for backend there are other step to fully run this project, REAME already provide inside backend folder)
3. npm run dev (to run this project)
